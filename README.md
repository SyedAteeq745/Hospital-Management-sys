# Patient-Centric Care System

This project showcases SQL coding within C# repositories for a patient-centric hospital management system. The focus is on improving patient engagement and care delivery.

---

## **Folder Structure**
- **CS_SQL**: Contains C# repository classes with SQL queries for database operations.

---

## **Repositories and Features**

### **1. PatientRepository**
- Manages SQL for `Patients` table:
  - Create Table
  - Insert Patient
  - Select Patients

### **2. AppointmentRepository**
- Manages SQL for `Appointments` table:
  - Create Table
  - Insert Appointment
  - Select Appointments with Patient and Doctor details

### **3. FeedbackRepository**
- Manages SQL for `Feedback` table:
  - Create Table
  - Insert Feedback
  - Select Feedback with Patient details

### **4. BillingRepository**
- Manages SQL for `Billing` table:
  - Create Table
  - Insert Billing Record
  - Select Billing Details for Patients
